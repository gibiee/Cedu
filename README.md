### problem
1. 유저코드(textarea)를 입력할 때 스크롤과 동시에 표시화면(div)도 스크롤 되어야함.
2. 표시화면(div)에 커서가 가면 안되서 none 속성을 주었더니 스크롤도 액세스가 안됨.

방법1. 유저코드(위) 표시화면(아래)
- 유저코드 투명도를 100%로 설정
- 너비를 잘 조절하면 표시화면의 스크롤을 이용할 수 있음.
- 유저코드의 스크롤 제거을 제거하면 서로 간에 연동이 안 되어 또 문제...

### 입력 이슈

왜냐면 ex. 숫자를 문자로 받을 이유가 없기 때문!
